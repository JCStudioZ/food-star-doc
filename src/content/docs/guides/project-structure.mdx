### Files and directories

- `.expo/` — Created when an Expo project is started using "expo start" command.
- `.storybook/` — Configuration files for Storybook UI that runs inside the native app.
- `.storybook_server/` — Configuration files for Storybook Server that runs on the browser that you can use to select a component in the native app to preview it.
- `assets/` — App logo and splash screen images.
- `src/` — Other source code and files (components, data, hooks, navigation, etc.) of the project.
